TrueSight AI

Description:

The project titled “TrueSight AI” is a smart detection system developed to identify and interpret AI-generated visual content. It is designed as an interactive and user-friendly web application that allows users to upload images or videos and receive immediate feedback on whether the content is authentic or artificially generated. At the core of TrueSight AI lies a deep learning model trained to recognize patterns and inconsistencies typical of synthetic media. The system supports several features aimed at improving both usability and interpretability — including graphical confidence indicators and visual heatmaps that highlight decision-making regions in the image.

Key Features:
  • Real-time detection of image and video authenticity using a trained deep learning model.
  • Visual explanations through confidence gauges and heatmaps.
  • Detection history logging and downloadable PDF report generation.
  • Designed for ease of use by journalists, educators, and researchers.
  • Scalable architecture for future enhancements and integrations.

Tech Stack:

  • Python – Backend & AI Logic
  • Streamlit – Web Interface Framework
  • HTML – Page Structure & Layout
  • CSS – Styling and Design
  • JavaScript – UI Behavior (internally via Streamlit)
